PO3 Style Guide

This is the style guide for IAT 339 PO3 - Portfolio.

Navigation Button

CSS selector:

.button-nav

Sample code:

<a href="#" class="button-nav">Nav Button</a>

Rendered element:

Nav Button

In-text link

CSS selector:

.in-text-link

Sample code:

<p>This is a in text link, <a href="#" class="in-text-link">Hello</a>, click to check!</p>

Rendered element:

This is a in text link, Hello, click to check!

Submit/Reset Buttons

CSS selector:

.button-submit

Sample code:

<button class="button-submit">Submit/Reset</button>

Rendered element:

Text Input Field

CSS selector:

.input-box

.input-label

Sample code:

<label class="input-label">Sample Text Input:</label><input class="input-box" type="text">

Rendered element:

Headings

CSS selector:

.heading-lv1

.heading-lv2

.heading-lv3

Sample code:

<h1 class="heading-lv1">Header Level 1</h1> <h2 class="heading-lv2">Header Level 2</h2> <h3 class="heading-lv3">Header Level 3</h3>

Rendered element:

Header Level 1

Header Level 2

Header Level 3

Paragraphs

CSS selector:

.text-p

Sample code:

<p class="text-p">This is a paragraph.</p>

Rendered element:

This is a paragraph. Morbi tristique senectus et netus et malesuada fames ac turpis egestas. Vestibulum tortor quam, feugiat vitae, ultricies eget, tempor sit amet, ante.

Bulleted Lists

CSS selector:

.ul-list

Sample code:

<ul class="ul-list"></li>List item 1</li></ul>

Rendered element:

  • List item 1
  • List item 2
  • List item 3

Numbered Lists

CSS selector:

.ol-list

Sample code:

<ol class="ol-list"></li>List item 1</li></ol>

Rendered element:

  1. List item 1
  2. List item 2
  3. List item 3

Main Navigation

CSS selector:

.nav-main

Sample code:

<div class="nav-main"> <a href="#">Home</a> <a href="#">About Us</a> <a href="#">Products</a> <a href="#">Contact Us</a> <a href="#">Account Management</a> </div>

Rendered element:

Home About Me Portfolio Experience Contact
